## Project Overview
This repository contains the analysis and findings from the **Social Media Marketing** project completed as part of the curriculum at WorldQuant University. The project applies data science methodologies to marketing datasets to extract actionable insights, evaluate campaign ROI, and understand audience behavior through statistical rigor.

---

## Key Features
* **Exploratory Data Analysis (EDA):** Deep dive into marketing datasets to identify correlations between ad spend, reach, and engagement.
* **Performance Metrics:** Calculation and visualization of critical KPIs, including **CTR (Click-Through Rate)**, **CPC (Cost Per Click)**, and **Conversion Rates**.
* **Statistical Modeling:** Using regression analysis to predict future engagement based on historical performance.
* **Data Visualization:** Comprehensive charts and heatmaps highlighting peak engagement times and demographic distributions.

---

## Tech Stack
* **Language:** Python
* **Libraries:** * `Pandas` for data manipulation
    * `NumPy` for numerical analysis
    * `Matplotlib` & `Seaborn` for statistical visualization
    * `Scikit-learn` for predictive modeling
* **Environment:** Jupyter Notebook

---

## Analysis Workflow
1. **Data Cleaning:** Handling missing values and normalizing marketing spend across different platforms.
2. **Feature Engineering:** Deriving new metrics such as "Engagement Velocity" and "Customer Acquisition Cost (CAC)."
3. **Hypothesis Testing:** Conducting A/B testing analysis to determine the statistical significance of different marketing creatives.
4. **Conclusion & Recommendations:** Summarizing data-backed strategies for optimizing digital marketing budgets.
